# Sales Data Dashboard

This project is an interactive **Sales Data Dashboard** created using **Microsoft Excel**.

The dashboard helps analyze sales performance using different filters, charts, and slicers. It provides useful insights about sales based on product line, country, month, year, and order status.

---

## Features

- Interactive slicers for filtering data
- Monthly sales trend analysis
- Sales comparison by product line
- Sales distribution by country
- Average sales analysis by country and order status
- Year-wise sales analysis
- Product line-wise sales analysis
- Country-wise sales analysis

---

## Dashboard Components

### 1. Monthly Sales Trend by Product Line
Shows monthly sales performance for different product categories such as:

- Classic Cars
- Motorcycles
- Planes
- Trucks and Buses
- Vintage Cars

### 2. Sum of Sales by Product Line and Year
Displays total sales distribution across different product lines and years.

### 3. Total Sales Distribution by Country
Shows how total sales are distributed among different countries.

### 4. Average Sales by Country and Status
Compares average sales based on country and order status.
